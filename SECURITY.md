# Security Policy

## Reporting a Vulnerability

If you discover a security vulnerability in FED, please report it responsibly:

1. **DO NOT** open a public GitHub issue
2. Contact **AKLGU** directly on Discord: @assttz
3. Or join the [FED Support Server](https://discord.gg/jWb5ArnHk6) and open a private ticket

We take security seriously and will respond within 48 hours.

## Scope

This policy applies to:
- The FED Discord bot
- The FED VPS infrastructure

## What qualifies as a vulnerability:
- Authentication or authorization bypass
- Data exposure or leakage
- Remote code execution
- Privilege escalation
- Denial of service

## What does NOT qualify:
- Social engineering attacks
- Attacks requiring physical access
- Issues in third-party dependencies (report those to the dependency maintainers)
- Theoretical attacks without proof of concept
